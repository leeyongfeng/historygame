# 华夏朝代配对闯关

国风历史卡牌配对游戏（单文件静态网页，零依赖）。

- 玩法：朝代↔事件点击配对、历史时间线排序、章节闯关、限时速配。
- 直接双击 `index.html` 即可本地游玩；或部署到任意静态托管（GitHub Pages / Vercel / Netlify）。
- 图片位于 `assets/`，与 `index.html` 同级。

## 本地预览
```bash
# 任选其一
python3 -m http.server 8080   # 然后访问 http://localhost:8080
# 或直接双击 index.html
```

部署到 GitHub Pages：把本目录内容推到仓库，在仓库 Settings → Pages 选择对应分支的根目录即可。
